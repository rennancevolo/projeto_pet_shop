# 🐾 Empório dos Bichos

Website institucional para petshop, desenvolvido para apresentar a empresa, serviços, localização e facilitar o contato com clientes.

## 📌 Sobre o projeto

O **Empório dos Bichos** é uma plataforma web criada para fortalecer a presença digital do petshop, oferecendo uma navegação simples, moderna e intuitiva para os clientes.

O objetivo principal é centralizar informações importantes da empresa e facilitar o acesso aos serviços e canais de atendimento.

## ✨ Funcionalidades

- Página inicial institucional
- Sessão "Sobre nós"
- Catálogo de serviços
- Informações de contato
- Botão direto para WhatsApp
- Integração com Instagram
- Localização via Google Maps
- Horário de funcionamento
- Layout responsivo
- Interface moderna e minimalista

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Responsive Design
